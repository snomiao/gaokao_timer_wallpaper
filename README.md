# 高考倒计时 / 桌面壁纸

起初是放到班上的多媒体设备，来给大家提高士气的。。
后来发现好像自己用更好的说……

语言环境：aardio

## 目前功能：
显示倒计时以及自定义信息
并用红色标出倒计时天数

启动后每分钟自动刷新壁纸

并没有自动设为开机启动，需要自己拖到启动
按`win`+`R`，输入 `shell:common startup`，即可打开自启动文件夹，拖进去就好

## 显示：
目标学校 现在时间
高考倒计时天数
